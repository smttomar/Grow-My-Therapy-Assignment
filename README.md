# Therapy Website ‚Äì UI Clone & Redesign Assignment

This project is a frontend-focused assignment where a therapy website homepage was first **cloned for UI accuracy**, then **redesigned using an imaginary therapist profile**, and finally enhanced with **custom UX interactions**.  
The goal was to demonstrate attention to detail, design thinking, responsiveness, and clean implementation using modern tools.

## üìå Project Breakdown

### **Part 1: Homepage UI Clone**

- Cloned the homepage layout from the provided template
- Matched spacing, typography, grid system, and section order
- Ensured full responsiveness (desktop, tablet, mobile)
- Used reusable Tailwind utilities and consistent spacing

---

### **Part 2: Redesign Using Therapist Profile**

- Redesigned the homepage using **Dr. Maya Reynolds (imaginary therapist)** as the single source of truth
- Introduced a new cohesive color palette and font system
- Rewrote all homepage copy (hero, services, about, FAQs) based strictly on the profile
- Replaced all images with intentional, calm, therapy-appropriate visuals
- Optimized copy for local SEO (Santa Monica, CA)
- Maintained readability, contrast, and visual balance

---
